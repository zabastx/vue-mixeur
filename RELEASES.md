# Release Notes v0.27.1

**Released**: May 19, 2026

## Bug Fixes

- **Project Export**: Fixed an issue where the active render camera was not being saved to `.mixeur` project files. Your camera selection is now preserved when saving and reopening projects.
- **Object Cloning**: Fixed a bug where exporting objects could cause unintended side effects. Objects are now properly cloned before export to prevent scene mutations.

For the complete list of changes, see the [Changelog](CHANGELOG.md).
